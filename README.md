## Instalar El Bot
Pasos para instalar el bot en termux

```bash
> pkg install git

> pkg install wget
> git clone https://github.com/GT-X1/ATX
> cd ATX
> bash install.sh
> npm start
> Y escaneas el QR con tu Whatsapp
```
